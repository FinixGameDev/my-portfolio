Hi! I'm Diogo Rocha aka Finix, and this is my portfolio. I'm a young game developer and game designer from Portugal and I recently graduated from IPCA studying Digital Games Development & Engineering.

If you're here, chances are you have heard my story a thousand times: I started my gaming journey pretty much in diapers, playing Max Payne on my dad's laptop, I then found my calling to game development at 14 when my middle school friends introduced me to the Unity Engine and so I've been learning about game dev and loving the medium ever since.

Anyways... here you can find some of the games and game development tools I've developed on my own, and some of the projects I had the pleasure of having a team to work with.

Academic Projects

Bean War

A small on-line shooter prototype made in Unity with Proton for networking. The game features a procedural generated circular maze as a map where up to 16 players can shoot each other in a free-for-all. Missing features include, a non-working leader board and ammo count.

As Aventuras de Lexi - Prototype

Made for a co-creation project with a multi-disciplinary team. As Aventuras de Lexi or Lexi's Adventures is an educational game design to help dislexic kids and children with other types of learning disabilities. I was responsible for the game design, level design, programming and writing as well of the original prototype which proceeded to gain some notoriety.

Maze Generator Tool

A Unity dev tool designed for Sam's Last Nightmare to create procedural generated levels with a maze format. Based on the book Mazes for Programmers, this tool supports sidewinder, binary tree and recursive backtracker algorithms for the map data generation as well as two instatiation methods. For gameplay reasons it also supports modifiers that change tiles based on the game rules.

Escape Rkhanspri

A smal tech demo project made for a shaders unit, Escape Rkhanspri intends to create a PSX-Retro style horror experience with a night vision shader, a pixelated shader, a noise shader that reacts when enemies are nearby and custom water shader with a dynamic transparency and vertex displacement.

My first ever game jam project. Made for

GameJam+ 2021
with the theme "Colors". The Sniffer is game about a paint sniffer that can go through walls as long as the paint they snif matches the wall color. I contributed mainly for the art, game design and level design for the game.

Love & Darkness

Made for Retro Jam 2022 with the theme of conections, Love and Darkness features two entities that are tethered together and in oposition. When one dies the other revives and the player is tasked with turning on four lighthouses in order to unite these entities. I contributed with the game design, level design and ""art".

RUN

Made for GMTK 2023 with the theme of roles reversed, RUN is about your run of the mill horror game mob, going about their life spooking players, until, a speedrunner on a genocide 100% run comes in. As such you need to run away from the speedrunner. This project was my first venture in Godot and is very much rudementary, featuring simple tank controls and a chase AI in a simple square. I contributed with the gameplay controls, AI and Menus.
